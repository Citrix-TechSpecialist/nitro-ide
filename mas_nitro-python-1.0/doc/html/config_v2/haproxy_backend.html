haproxy_backend

Configuration for HAProxy Backend resource.

Read/write properties

id<String>

Id is system generated key for all the HAProxy Instances.

Read only properties

algorithm<String>

HAProxy backend Algorithm ..

instance_id<String>

Instance ID of HA Proxy Instance..

configuration_path<String>

Config Path of HA Proxies on this HAPRoxy Host..

name<String>

HAProxy backend name ..

ip_address<String>

HAProxy host ipaddress..
