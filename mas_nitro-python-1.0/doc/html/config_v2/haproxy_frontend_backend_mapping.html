haproxy_frontend_backend_mapping

Configuration for HAProxy Frontend backend mapping resource.

Read/write properties

id<String>

Id is system generated key for all the HAProxy frontend backend mapping.

Read only properties

instance_id<String>

Instance ID of HA Proxy Instance..

configuration_path<String>

Config Path of HA Proxies on this HAPRoxy Host..

backend_name<String>

HAProxy backend name ..

ip_address<String>

HAProxy host ipaddress..

condition<String>

HAProxy frontend backend mapping condition ..

frontend_name<String>

HAProxy frontend name ..
