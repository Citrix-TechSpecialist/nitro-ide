syslog_messages

Configuration for System Log Message resource.

Read/write properties

source<String>

Source IP Address.

event_type<String>

Event type.

message<String>

Message.

app_name<String>

Application Name.

system_time<String>

System Time in local timezone.

id<String>

Id is system generated key for all the system log entries.

instance_ip<String>

Instance IP Address.

severity<String>

Severity of the message.

module<String>

Module name.

Read only properties

system_gmt_time<Integer>

System Time in GMT timezone.

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE

filter

Use this query-parameter to get the filtered set of syslog_messages resources configured on the system. You can set a filter on any property of the resource.

pagesize=#no&pageno=#no

http://<MGMT-IP>/nitro/v2/config/syslog_messages?pagesize=#no&pageno=#no

Use this query-parameter to get the syslog_messages resources in chunks.

count

http://<MGMT-IP>/nitro/v2/config/syslog_messages?count=yes

Use this query-parameter to get the count of syslog_messages resources.
