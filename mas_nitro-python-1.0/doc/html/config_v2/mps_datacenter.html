mps_datacenter

Configuration for MPS Datacentre resource.

Read/write properties

tenant_name<String>

Tenant Name.

longitude<Double>

longitude of the city.

is_default<Boolean>

IS Default Datacentre.

location<String>

Cloud Provide.

name<String>

Datacentre Name.

tenant_id<String>

Tenant ID.

device_type<Integer>

device_type.

latitude<Double>

latitude of the city.

cloud_provider<String>

Cloud Provide.

ip_block<String>

ID for IP Block.

type<Integer>

Datacenter type: Site/Datacenter.

id<String>

Id is system generated key for all the system users.

device_ip<String>

device_ip.

agent_node_list_array<String[]>

Array for Agent nodes.

ip_block_array<String[]>

Array for Devices.

cityname<String>

City Name.
Maximum length = 255

count_map<entity_map[]>

Count Map.

clients<client_connection[]>

clients.

server_node_list_array<String[]>

Array for Server nodes.

servers<server_connection[]>

Count Map.

device_list_array<String[]>

Array for Devices.

Read only properties

total_critical_events<Integer>

Total Critical events city wise..

total_major_events<Integer>

Total major events..

total_events<Integer>

Total events city wise..

total_number_of_devices<Integer>

Total number of devices deployed city wise...

total_number_of_agents<Integer>

Total number of agents deployed city wise..
