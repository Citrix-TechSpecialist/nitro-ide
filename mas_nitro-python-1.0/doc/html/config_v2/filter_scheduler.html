filter_scheduler

Configuration for Filter Scheduler resource.

Read/write properties

details<filter_schedule_details[]>

Filter Schedule Details.

recurrence_type<String>

Tenant.
Maximum length = 255

tz_offset<Integer>

Time zone offset..

filter_job_name<String>

Filter Job Name.
Maximum length = 255

recurrenceOptions<String>

Comma Seperated recurrence options of job that is scheduled.

id<String>

Id is system generated key for all the events.

filter_status<String>

Device Group Criteria.
Maximum length = 255

Read only properties

activity_log<String>

Device Group Criteria.

activity_status<String>

Activity Status.
