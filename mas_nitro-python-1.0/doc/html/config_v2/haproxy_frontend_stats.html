haproxy_frontend_stats

Configuration for HAProxy frontend stats information resource.

Read/write properties

bin<Double>

Bytes of date in.

hrsp_1xx<Double>

No of request with response code 1XX.

bout<Double>

Bytes of data out.

hrsp_other<Double>

No of request with other response code.

dresp<Double>

No of response denied.

scur<Double>

Current Session.

id<String>

Id is system generated key.

hrsp_2xx<Double>

No of request with response code 2XX.

dreq<Double>

No of denied requests.

frontend_id<String>

Frontend ID.
Minimum length = 1
Maximum length = 64

svname<String>

Service Name.
Minimum length = 1
Maximum length = 100

hrsp_4xx<Double>

No of request with response code 4XX.

hrsp_3xx<Double>

No of request with response code 3XX.

hrsp_5xx<Double>

No of request with response code 5XX.

Read only properties

req_rate<Double>

HTTP Reqrate/Sec.

ereq<String>

Number of error requests.

req_tot<String>

Total number of requests.

eresp<String>

Number of error responses.

stot<Double>

Total number of sessions.

Default value: EXIT
Possible values = EXIT, CONTINUE
