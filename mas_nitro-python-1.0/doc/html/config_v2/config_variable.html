config_variable

Configuration for Configuration Variables for use in configuration template resource.

Read/write properties

parent_name<String>

.

name<String>

Variable name.
Minimum length = 1
Maximum length = 128

display_name<String>

Display name of configuration variable.
Minimum length = 1
Maximum length = 1024

description<String>

Description of configuration variable.
Minimum length = 1
Maximum length = 1024

parent_id<String>

.

required<Boolean>

If it is required argument for command then true else false..

default_value<String>

Default Value of configuration variable.

id<String>

.

type<String>

Type of Variable.
Minimum length = 1
Maximum length = 64

values_enum_db<String>

Comma seperated list of possible values of variable.

values_enum<String[]>

Possible Values Enum for the configuration variable.

Read only properties
