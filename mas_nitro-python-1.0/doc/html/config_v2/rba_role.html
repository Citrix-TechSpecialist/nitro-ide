rba_role

Configuration for RBA Role resource.

Read/write properties

tenant_id<String>

Tenant Id of the RBA roles.
Minimum length = 1
Maximum length = 128

name<String>

Role Name.
Minimum length = 1
Maximum length = 128

id<String>

Id is system generated key for all the RBA roles.

description<String>

Description of Role.
Minimum length = 1
Maximum length = 1024

resourcegroups<String[]>

Resourcegroups attached to this role..

groups<String[]>

Groups to which this role is assigned.

policies<String[]>

Policies attached to this role..

Read only properties
