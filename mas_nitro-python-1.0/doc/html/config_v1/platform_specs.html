platform_specs

Configuration for Platform Specification resource.

Read/write properties

Read only properties

is_fips_present<Boolean>

To check if FIPS is present.

contains_fortville_nics<Boolean>

check if Fortville nic card is present.

max_channels<Integer>

Maximum number of channels available.

max_cores_per_vm<Integer>

Maximum number of CPU cores which can be allocated to VM.

is_nextgen<Boolean>

If NextGen Platform.

no_of_mgmt_interfaces<Integer>

total number of management interfaces available.

contains_raid<Boolean>

Enable RAID for the platform.
